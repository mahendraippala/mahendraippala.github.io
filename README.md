# Mahendra Reddy Ippala - Portfolio

A modern, responsive portfolio website showcasing my work as a Cloud Solutions Architect & Engineer.

## 🚀 View Live

## 📁 Project Structure

```
portfolio/
├── index.html    # Main HTML file
├── styles.css    # All styling and animations
└── script.js     # Interactive functionality
```

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom properties, animations, flexbox, grid)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter, Fira Code)

## ✨ Features

- Responsive design (mobile, tablet, desktop)
- Smooth scroll animations
- Typing effect for titles
- Animated cloud illustration with floating icons
- Skill bars with progress animation
- Interactive project cards
- Contact form with email integration
- Dark theme with gradient accents

## 📦 Deployment

This portfolio is hosted on GitHub Pages.

### To deploy your own:

1. Create a new repository on GitHub
2. Push these files to the repository
3. Go to Settings > Pages
4. Select main branch as source
5. Your site will be live at `https://yourusername.github.io/repository-name`
